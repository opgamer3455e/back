API Integration

RESTful

Integrate Aura directly into your pipeline. Send a multipart form request and receive a compressed ZIP archive stream.

curl -X POST http://localhost:3000/api/convert \
  -F "video=@./footage.mp4" \
  -F "fps=5" \
  -F "format=jpg" \
  -F "width=1280" \
  -o frames.zip
Endpoint POST /api/convert
Content-Type multipart/form-data
Response application/zip
